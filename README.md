## 🚀 Projetos em Produção

### 🏠 My Home Cuidadores — Sistema Completo de Gestão
Sistema digital completo para agência de cuidadores de idosos — do site ao CRM ao bot de atendimento.

**O que foi construído:**
- **CRM customizado** com React + Supabase — gestão de clientes, cuidadores e plantões
- **Site institucional** servido em VPS própria com Caddy
- **Bot WhatsApp** (Node.js + Evolution API) — atendimento automatizado, notificações de plantão 4x/dia
- **Agente de Google Ads** com scripts automáticos — monitoramento e relatórios via Telegram
- **Edge Functions** no Supabase para notificações de turno em tempo real
- **Dashboard Mission Control** — painel unificado de monitoramento de toda a operação

**Stack:** React, Node.js, Supabase, Evolution API, Caddy, Ubuntu VPS, systemd, Claude API

### 🧠 Reviva — App de Reabilitação Pós-AVC
App de estimulação cognitiva e de linguagem para pacientes em recuperação de AVC. Criado a partir de experiência pessoal — meu pai passou por dois AVCs.

**O que foi construído:**
- App web com 7 módulos de exercícios cognitivos e de linguagem
- Banco de dados no Supabase com banco de palavras e hints semânticos
- Integração com TTS (síntese de voz) para exercícios de fala
- Interface projetada para uso autônomo por pacientes com mobilidade reduzida

**Stack:** React Native/Expo, Supabase, TTS API, Node.js

### 🤖 Booh — Agente de IA Operacional (COO da Infinity)
Sistema de agente de IA que opera como COO da empresa — memória persistente, múltiplos sub-agentes especializados, crons autônomos.

**O que foi construído:**
- OpenClaw configurado do zero em VPS Ubuntu
- Memory Palace — sistema de memória estruturada com busca semântica (nomic-embed-text + Ollama)
- 4 agentes especializados: Julia (marketing), Gabriel (dev), Ana (ops), Marcos (Google Ads)
- Integração Telegram como interface principal de controle
- Fallback chain de modelos: Claude Sonnet → Gemini Flash → Ollama local (gemma4:26b)
- Tailscale VPN conectando VPS de produção ↔ servidor local com GPU

**Stack:** OpenClaw, Claude API, Gemini API, Ollama, Tailscale, Telegram Bot API, systemd
